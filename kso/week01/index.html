项目结构

week01.sln 解决方案中包括3个项目：ReverseKGroup、UserInterface和Test。

ReverseKGroup 项目 生成静态库 ReverseKGroup.lib，其中包括按组反转链表的核心算法。

UserInterface 项目 生成可执行文件 week01.exe ，链接至 ReverseKGroup.lib。它为算法提供命令行界面，方便用户交互。

Test 项目 生成可执行文件 Test.exe，链接至 ReverseKGroup.lib。它是单元测试模块。

从源代码构建

  1. 用 Visual Studio 2019 打开解决方案文件 week01.sln。
  2. 在菜单栏中选择 生成 -> 生成解决方案
  3. 在 yangerhang\week01\x64\Debug\ 中可以找到 week01.exe。双击即可运行。

运行单元测试

  1. 在菜单栏中选择 测试 -> 运行所有测试
  2. 在菜单栏中选择 测试 -> 测试资源管理器，可以打开测试资源管理器，查看测试运行情况

文件模式

在命令行运行.\week01.exe <文件路径>将以文件模式启动本程序。

举例来说，假设输入文件 testcases.txt 的内容如下：

head = [1,2,3,4,5], k=2
head = [1,2,3,4,5], k=3
head = [1,2,3,4,5], k=1
head = [1], k=2
head = [1], k=1
