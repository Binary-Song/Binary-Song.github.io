详细描述

template<typename T>
class kso::String< T >::ConstIterator

常迭代器。不能通过本迭代器更改String内部元素的值。

注解
本类型的所有函数都通过类型转换映射到 String<T>::Iterator 类的对应函数 目的是避免维护重复代码 （改了一个忘记改另一个）

以下使用的所有强制类型转换都是安全的。C++ 标准规定， 在standard layout 的前提下，指向对象的指针也一定指向它的第一个成员变量。 （C语言也是如此），例如： struct A {int x;}; A a; 则一定有 (void*)&a == (void*)&(a.x)，而强制类型转换是不改变指针的值的。 所以我们只要保证 ConstIterator 和 Iterator 是 standard layout， 即可确保 ConstIterator* 和 Iterator* 的互转是安全的。
