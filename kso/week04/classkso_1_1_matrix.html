详细描述

template<typename T>
class kso::Matrix< T >

矩阵。本类支持copy-on-write，在不修改的前提下，拷贝的开销很低。

模板参数
T标量类型

copy-on-write的机制： Matrix 类不是“数据”，而是数据的“视图”。 底层实现为 std::shared_ptr。每个矩阵对象都对数据构成强引用。当某块数据 不被任何矩阵对象引用时，它才会被回收。 当拷贝本类对象时，并不会立即拷贝数据，而是先浅拷贝 std::shared_ptr 指针，从而增加引用计数。 当对数据做修改时（参见“修改”区域），会检查引用计数，如果大于1（数据非独有）， 就拷贝一份再修改。

创建矩阵：

// 创建2*2的全8矩阵
Matrix<int> m(2, 2);
m.fill(8);
std::cout << m;

访问元素：

// 将(0,1)位置设置为15
m.set(0,1,15);
// 访问(0,1)位置的元素
std::cout << m.get(0,1);

按行遍历：

for(auto&& i : m)
std::cout << i << " ";

按列遍历：

for(auto&& i : m.transpose())
std::cout << i << " ";

矩阵乘法（点乘）：

Matrix<int> m1(2,3,{1,2,3,4,5,6});
Matrix<int> m2(3,2,{1,2,3,4,5,6});
Matrix<int> m3 = m1.dot(m2);
std::cout << m3 << std::endl;
